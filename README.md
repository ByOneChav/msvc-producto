# 🧴 Producto Service

![Producto](https://source.unsplash.com/800x400/?perfume,cosmetics,product)

Administra el catálogo de productos.

## 🚀 Funcionalidades

- CRUD productos
- Consulta de stock
- Integración con Inventario

## 🛠️ Tecnologías

- Spring Boot
- JPA
- Oracle SQL

## 🔗 Endpoints

- `POST /productos`
- `GET /productos`
- `GET /productos/{id}`
- `PUT /productos/{id}`
- `DELETE /productos/{id}`
